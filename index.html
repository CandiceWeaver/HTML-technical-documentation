Basic HTML
Hello World

Most HTML elements have opening and closing tags.

Opening tags look like this:

<h1>

Closing tags look like this:

<⁄h1>

The only difference between the two tags is the "⁄" after the opening bracket of the closing tag. As a convention, all HTML tags are written in lowercase rather than uppercase.

Heading Elements

h1 elements are used for main headings on a website and h2 are usually used for subheadings. Different levels of subheadings are indicated by h3, h4, h5, and h6 elements.

Paragraph Elements

p elements are used for paragraph text on websites. The p itself is short for paragraph. You can create paragraph elements like this:

<p>Hello World<⁄p>
Placeholder Text

Developers traditionally use lorem ipsum as placeholder text. Lorem ipsum is text that has been randomly scraped from a famous passage by Cicero of Ancient Rome. It has been used as placeholder text since the 16th century and the tradition still continues on the web today. These days, however, there are many different types of lorem ipsum text, many of which can be found here.

Comments in HTML Elements

Commenting is a way to leave comments for other developers within your code without being displayed to the end user. It is also a way of making code inactive without deleting it. Comments in HTML start with <!–– and end with ––>.

HTML5 Elements

HTML5 introduces more descriptive tags. Some of these include main, header, footer, nav, video, article, and section. These tags help give a descriptive structure to HTML, make HTML easier to read, Search Engine Optimisation (SEO), and accessibility. For example, the main tag helps search engines and developers find the main content of the page.

Image Elements

Images can be added to a website by using the img element. You can point to a specific URL or file using the src attribute. img elements are self closing, so does not need the closing tag.

All img elements must also have an alt attribute. The text inside this attribute is used to improve accessibility and is displayed if the image in unable to load. If the image is only for decoration, then the alt attribute can be left empty. It should not contain any special characters if possible.

An example of this element would be:

<img src="https://images.unsplash.com/photo-1556582305-528bffcf7af0?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxzZWFyY2h8MTZ8fGN1dGUlMjBjYXR8ZW58MHx8MHx8&auto=format&fit=crop&w=500&q=60" alt="A cute kitten sleeping">
Anchor Elements

You can use anchor elements to link to content both within and outside of a web page. a elements need a href attribute that contains the destination web address or file location. They also need anchor text. Here is an example:

<a href="https://example.com" target="_blank">This is a link<⁄a>

With the above code, your browser with display This is a link as a link you can click on. This link with take you to the URL https://example.com. The target="_blank" attribute will make the link open in a new tab.

You can also use the anchor element to create internal links to different sections within a web page. To do this, you need to assign the href link to the id of the element you want to link to, starting with the # symbol. You need to make sure the element you are linking to has that id. An id attribute uniquely describes an element. Please see the example below:

<a href="#about-section">About<⁄a>
. . .
<h2 id="about-section">About<⁄h2>

In this example, users will be taken to the section of the web page with the About heading element when they click on the About link.

Sometimes you want to add anchor elements to a website before you know where they will link. In this case, just use a # as the href value. This can be handy when using JavaScript too.

You can turn other elements into links by nesting them within an anchor element, just as images.

Unordered Lists

HTML has an elements for creating unordered lists, such as bullet point style lists. These lists start with an <ul> element followed by at least 1 <li> element. You close unordered list elements with <⁄ul>.

<ul>
 <li>bread<⁄li>
 <li>butter<⁄li>
<⁄ul>

The above code will display an unordered list like this one:

  • bread
  • butter
Ordered Lists

HTML also has an element for creating ordered lists, such as numbered lists. These lists start with an <ol> element followed by at least 1 <li> element. You close unordered list elements with <⁄ol>.

<ol>
 <li>bread<⁄li>
 <li>butter<⁄li>
<⁄ol>

The above code will display an unordered list like this one:

  1. bread
  2. butter
Input Elements

input elements are convenient ways to recieve inout from the user. These elements are self-closing and look like this:

<input type="text">

The type attribute defines what type of input the element is. input elements are self-closing.

Placeholder text is the text displayed in the input element before the user has inputted any data. Placeholder text is written as follows:

<input type="text" placeholder="this is placeholder text">

You can mark form fields as required, which forced the user to fill in that form field before submitting the form. You just need to add required within the input tag.

Form Elements

You can build forms that submit data to a server just using HTML. You do this by defining an action attribute on the form element.

<form action="the-url-where-you-want-the-form-to-be-submitted">
 <input>
<⁄form>
Submit Buttons

The submit button will send your form data to the URL specified in the form's action attribute. It looks like this:

<button type="submit">Submit<⁄button>
Radio Buttons

You can use radio buttons for form inputs where you want the user to choose just one choice out of the possible options. Just set the type as type="radio".

Each radio button can be nested within its own label element. By doing this, the label element will automatically associate the radio button input with the surrounding label element.

All related radio buttons should have the same name attribute to create a group of radio buttons. This means that selecting a radio button will automatically unselect the other buttons in the group so that only one option can be selected by the use.

<label for="radio-button">
 <input type="radio" id="radio-button" name="example" checked>Radio button
<⁄label>

It is best practice to set a for attribute on the label element, whose value matches the id value of the input element. This lets assistive technologies to understand the linked relationship between the label and the input element.

You can set a radio button to be selected by default by using the checked attribute as shown above.

Checkboxes

Forms often use checkboxes for inputs where the user can select more than one option. They are a type of input and are treated the same asradio buttons, except the type attribute is checkbox.

The div Element

The div element is also known as the division element. It is used as a general container for other elements and is one of the most used elements in HTML. You can open the element with <div> and you can close it with <⁄div>

Declaring the Doctype

This element give a web page structure and should be included in every HTML document. At the beginning of the document, you need to tell the browser which version of HTML you are using. HTML is a constantly evolving language and is updated regularly. While most browsers support the lastest version (HTML5), some older pages may use previous versions of HTML.

You can tell the browser this information by adding the <!DOCTYPE ...> tag on the first line. The ... is where you write the version of HTML being used. For HTML5, you use <!DOCTYPE html>. Everything in this tag except for html is case sensitive and must be written in capitals, especially for older browsers.

The rest of the HTML document has to be wrapped in html tags. The opening tag goes after the Doctype and the closing tag goes at the end of the document. Your HTML code would go between the html tags as shown below:

<!DOCTYPE html>
<html>
 <!-- Your code goes here -->
<⁄html>
Head and Body Elements

These tags are used to make a HTML document more organised. They go between the html tags. Metadata elements go in the HEAD tag. Metadata elements include elements such as link, meta, title, and style

Any content for the page goes in the body tag.

<html>
